Week 3 Session 4 Class Work Tasks - Java Full Stack Development (JavaFSD)

This repository contains the solutions and SQL queries completed during Week 3 Session 4 of the Java Full Stack Development (JavaFSD) training. The tasks cover various SQL operations on an Employee database, including:

INSERT Operations
UPDATE Operations
DELETE Operations
SELECT Queries
WHERE, AND, OR, NOT Conditions
ORDER BY (ASC & DESC)
DISTINCT
LIMIT
BETWEEN
IN
LIKE
Aggregate Functions (COUNT, SUM, AVG, MAX, MIN)

The objective of these exercises is to strengthen SQL database management and query-writing skills through practical hands-on tasks using employee records.


Technologies Used:

SQL
Oracle Database
